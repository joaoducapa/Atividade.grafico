Entenda os Modelos de Cores RGB e CMYK

Explore a diferença entre os modelos de cor usados em telas e impressões.

O que são RGB e CMYK?

RGB (Red, Green, Blue) e CMYK (Ciano, Magenta, Amarelo, Preto) são dois modelos de cores amplamente utilizados em diferentes contextos. RGB é empregado para telas digitais, enquanto CMYK é voltado para impressões.

O que é RGB?

O modelo **RGB (Red, Green, Blue)** é baseado na adição de luz para criar cores. Ele é usado em **telas e monitores**, como TVs, smartphones e computadores.

As três cores primárias (Vermelho, Verde e Azul) se combinam em diferentes intensidades para formar todas as outras cores.

Quando os três componentes estão no máximo (255, 255, 255), o resultado é **branco**; quando estão no mínimo (0, 0, 0), o resultado é **preto**.

Alguns exemplos de combinações RGB:

  • rgb(255, 0, 0) → Vermelho
  • rgb(0, 255, 0) → Verde
  • rgb(0, 0, 255) → Azul
  • rgb(255, 255, 255) → Branco (combinação máxima de luz)
  • rgb(0, 0, 0) → Preto (ausência de luz)

O que é CMYK?

O modelo **CMYK (Ciano, Magenta, Amarelo, Preto)** é utilizado na impressão, onde as cores são formadas pela sobreposição de pigmentos.

Diferente do RGB, que usa luz para criar cores, o CMYK funciona pela **subtração de luz refletida**.

Quando todas as cores são combinadas no CMYK, o resultado teórico é preto. No entanto, na prática, um preto puro (K) é adicionado para melhorar a profundidade e o contraste.

Exemplos de misturas CMYK:

  • C + M + Y = Preto (na teoria, mas na prática um preto puro é adicionado para mais profundidade)
  • C + M = Azul
  • M + Y = Vermelho
  • Y + C = Verde

Comparação entre RGB e CMYK

Os dois modelos são usados para propósitos distintos:

  • RGB: Ideal para imagens digitais, telas e dispositivos eletrônicos.
  • CMYK: Indicado para impressões gráficas, pois trabalha com pigmentos.

Uma mesma cor pode ter uma aparência diferente dependendo do modelo usado. Por isso, designers precisam converter as cores corretamente entre RGB e CMYK ao criar materiais para impressão.

Conclusão

O modelo RGB é usado para dispositivos eletrônicos, enquanto o CMYK é essencial para impressões. Conhecer as diferenças entre esses modelos é crucial para garantir que as cores sejam representadas corretamente em cada tipo de mídia.
